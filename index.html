如何取得 Google OAuth Client ID

  1. 前往 Google Cloud Console（console.cloud.google.com）
  2. 建立新專案（或選既有專案）
  3. 左側 → API 和服務 → 搜尋並啟用 Google Drive API
  4. 左側 → 憑證 → 點「建立憑證」→ 選 OAuth 用戶端 ID
  5. 應用程式類型選 網頁應用程式（Web application）
  6. 在「授權的 JavaScript 來源」加入：https://mindyworldnews.github.io
  7. 建立後，複製 用戶端 ID（結尾為 .apps.googleusercontent.com）
  8. 貼到上方欄位並點「儲存 Client ID」

⚠️ 首次授權時 Google 可能顯示「未驗證的應用程式」，點「進階」→「前往（不安全）」即可繼續。

💡 Client ID 只是識別用，不是密碼，存在瀏覽器裡是安全的。
